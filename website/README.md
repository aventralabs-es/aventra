# Aventra Website MVP

Static MVP for the public Aventra website.

## Files

- `index.html` - single-page website with catalog, platform, pricing, education, Setup Radar, support, and legal sections.
- `styles.css` - responsive product-site styling.
- `app.js` - indicator catalog filtering.

## Local Preview

Open `index.html` directly in a browser. No build step is required for this MVP.
