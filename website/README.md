# Aventra Website

Next.js + TypeScript application for the public Aventra website and upcoming Market Radar product work.

## Stack

- Next.js App Router
- TypeScript
- Vercel hosting and Web Analytics
- Static public blog compatibility during migration

## Key Routes

- `/` - product homepage, indicator catalog, education, and Market Radar entry point.
- `/market-radar` - Market Radar architecture and roadmap.
- `/blog/*.html` - legacy education pages served from `public/blog` until they are migrated to native Next.js routes.
- `/robots.txt` and `/sitemap.xml` - generated by Next.js metadata routes.

## Local Development

```powershell
npm install
npm run dev
```

Production build check:

```powershell
npm run build
```
