# MetaTrader Guidance

## Scope

MetaTrader work should translate Aventra concepts into MT4/MT5 indicators and, later, expert advisors only when explicitly requested.

## Product Rules

- Keep names aligned with TradingView where possible.
- Public-facing strings should be English.
- Prefer MT5 first when building new premium-grade products.
- Keep licensing and marketplace constraints separate from indicator logic.

## Sales Notes

- If publishing through MQL5 Market, do not design the product around an external payment/license system unless the user explicitly chooses a non-Market distribution path.
- Use the Aventra website for documentation, product pages, support, and roadmap.
- Treat MQL5 Market as the likely primary sales and activation channel for MetaTrader products.
