# TradingView Publish Texts

This folder contains copy-ready text for TradingView publishing forms.

Use these files for the publish modal:

- `*_description.md`: paste into the TradingView description field
- `*_release_notes.md`: paste into the TradingView release notes field

Current publish-ready descriptions:

- `ma_crossover_signals_description.md`
- `dynamic_support_resistance_description.md`
- `trend_strength_matrix_description.md`
- `liquidity_zones_description.md`
- `volume_pressure_description.md`
- `market_structure_description.md`
- `fair_value_gaps_description.md`

These files are public-facing product descriptions. They are intentionally different from `indicators/tradingview/README.md`, which is internal project documentation.
