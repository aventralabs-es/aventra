# cTrader Guidance

## Scope

cTrader work should translate Aventra concepts into cTrader indicators/cBots using cTrader Automate/cAlgo conventions.

## Product Rules

- Keep names aligned with TradingView where possible.
- Public-facing strings should be English.
- Keep visual defaults consistent with TradingView products where practical.
- Prefer clear parameter names and conservative defaults.

## Sales Notes

- Treat cTrader Store as the likely primary sales and distribution channel for cTrader products.
- Use the Aventra website for documentation, product pages, support, and roadmap.
